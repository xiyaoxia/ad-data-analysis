# 广告大盘数据分析全流程
## 项目背景
为解决广告周报数据手动拉取的痛点，完成广告大盘数据的整合、清洗与可视化，实现数据自动化输出。

## 核心工作
1. **SQL取数**：多表关联提取DAU、广告收益、曝光次数、ecpm等核心指标，用`CASE WHEN`拆分广告类型/场景维度，处理空值保证数据准确性；
2. **Python清洗**：统一日期格式、处理异常值、计算衍生指标（环比增长率、单曝光收益），提升数据质量；
3. **BI可视化**：制作广告大盘效果仪表盘，展示DAU趋势、收益对比、曝光-收益相关性等核心维度。

## 技术栈
- SQL（CTE、CASE WHEN、多表关联、空值处理）
- Python（Pandas数据清洗、模拟数据生成）
- Power BI（可视化仪表盘）

## 项目价值
- 自动化周报取数，提升效率80%；
- 数据校验机制保证周报数据准确性；
- 可视化仪表盘辅助业务快速定位广告效果问题。
